## Project Description
This project focuses on RNA sequencing (RNA-seq) analysis. RNA-seq is a high-throughput sequencing technique used to measure and compare gene expression levels across different biological conditions. It helps identify differentially expressed genes, understand cellular responses, and study molecular mechanisms underlying diseases and biological processes.

## Project Structure
- data/: Contains raw, processed, and result datasets
- scripts/: Contains analysis scripts
- docs/: Contains documentation
